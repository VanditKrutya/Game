# Game

## What it does

A simple Browser game that shows generates two random dices. The player with the higher score wins. Draw if both roll the same number. Uses HTML, CSS and Vanilla JavaScript as frontend.

<br />

## Uses

- HTML
- CSS
- JavaScript
